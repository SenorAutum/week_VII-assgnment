## üìñ Project Overview
This repository contains a technical audit of the **COMPAS Recidivism Dataset**, a notorious case study in algorithmic bias. The goal of this project is to use data science tools to quantify racial disparities in risk assessment algorithms and propose ethical remediation strategies.

This work addresses the critical question: **"Does the algorithm make mistakes equally across different demographic groups?"**

## üìÇ Repository Contents
* **`fairness_audit.py`**: The Python script performing the data analysis, modeling, and bias calculation.
* **`AI_Ethics_Report.pdf`**: A comprehensive theoretical analysis including case study reviews (Amazon, Policing) and a healthcare policy proposal.
* **`bias_audit_chart.png`**: (Generated after running the script) Visual comparison of False Positive Rates.
